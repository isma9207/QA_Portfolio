Urban Scooter

Utilizando los requisitos del aplicativo diseñé casos de prueba para para garantizar el correcto funcionamiento del pedido de servicio de scooters por parte del usuario, desde el inicio de sesión hasta la entrega del vehículo para finalizar el servicio. Las pruebas realizadas fueron en plataformas web, utilizando navegadores como Opera y Chrome, y en móviles utilizando como herramientas Android Studio para la emulación de dispositivos. Accedí a la base de datos utilizando postgreSQL para cambiar status del servicio en sus diferentes etapas asi como la creación de usuarios y pedidos mediante solicitudes a la API a través de Postman para corroborar su correcto funcionamiento de acuerdo con lo estipulado en requisitos.
