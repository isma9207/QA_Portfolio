Descripción del Proyecto

Este proyecto consistió en realizar pruebas para garantizar el correcto funcionamiento de un aplicativo para solicitar la renta de scooters. Las actividades principales fueron:

  • Diseño y ejecución de casos de prueba basados en requisitos del sistema.
  • Pruebas en navegadores (Opera y Chrome) y en móviles mediante emuladores de Android Studio.
  • Validación de datos y cambios de estado en la base de datos utilizando PostgreSQL.
  • Envío y validación de solicitudes a la API mediante Postman.
